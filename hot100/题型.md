## 3. 滑动窗口 (Sliding Window)

**对应文件夹**：`数组/长度最小的子数组/`、`栈与队列/滑动窗口最大值/`

**理论基础**：
- `总结模版.md` - 滑动窗口模板

**相关题目文件夹**：
- `数组/长度最小的子数组/` - 209.长度最小的子数组、904.水果成篮
- `栈与队列/滑动窗口最大值/` - 239.滑动窗口最大值（使用单调队列）

**核心模板**：
```cpp
while (right < n) {
    更新窗口状态
    while (窗口不满足条件) {
        缩小窗口
    }
    更新答案
}
```

**学习路径**：
1. 理解滑动窗口的基本思想
2. 掌握固定窗口和可变窗口
3. 学习单调队列优化（滑动窗口最大值）

## 📖 学习建议

### 刷题顺序建议

1. **基础阶段**（1-2周）
   - 数组、链表、字符串基础操作
   - 哈希表、双指针、滑动窗口
   - 栈、队列基础

2. **进阶阶段**（2-3周）
   - 二叉树遍历和操作
   - 回溯算法
   - 贪心算法基础

3. **高级阶段**（3-4周）
   - 动态规划（从简单到复杂）
   - 图论基础（DFS/BFS、并查集）
   - 高级数据结构（堆、单调栈）

4. **综合阶段**（持续）
   - 图论高级算法（最短路径、最小生成树）
   - 复杂DP问题
   - 综合技巧应用

### 学习方法

1. **理论学习**：先看理论基础文件（`.md`文件）
2. **模板练习**：掌握核心模板（如滑动窗口、回溯三部曲）
3. **题目练习**：按文件夹分类刷题
4. **总结归纳**：每类题型刷完后总结规律

### 重要提示

- **图论**：理论基础文件最全面，建议系统学习
- **动态规划**：从简单到复杂，循序渐进
- **回溯**：掌握模板后，主要练习去重技巧
- **贪心**：注意证明贪心策略的正确性
